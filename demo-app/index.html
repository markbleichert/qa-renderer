This is a test page for the qa runtime.

Test data set is embedded in the source of this page by script tag
This the runtime container. This text should not be visible..

The runtime will use the embedded data and render the first item as question with corresponding answers. When a question is ansered the next question is displayed, replacing the previous one.

Once an answer to a question has no follow up question an endpoint is reached. An enpoint is always a piece of content in the form of an article or a link to another page.
